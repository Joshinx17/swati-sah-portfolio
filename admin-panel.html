🚀 How Publishing Works

This admin panel stores all your changes locally (in your browser). To push changes live, you export a content.json file and commit it to your GitHub repository. GitHub Pages auto-deploys in ~60 seconds.

1
Make all your edits using the sidebar tabs
2
Click "⬇ Download content.json" above — save it to your computer
3
In your GitHub repo, navigate to the root folder and upload/replace content.json
4
Commit with a message like "Update content via ScholarCMS"
5
GitHub Pages auto-deploys — your live site updates in ~60 seconds ✅
