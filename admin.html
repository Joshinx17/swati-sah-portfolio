🚀 How to Publish

This admin panel saves all changes locally in your browser. To publish to your live site, download the JSON file and add it to your GitHub repository.

1
Make all your edits using the sidebar tabs
2
Click ⬇ Download content.json in the top bar
3
In your GitHub repo, go to the root folder and upload/replace content.json
4
Commit with message like "Update content via admin panel"
5
GitHub Pages will auto-deploy — your site updates in ~60 seconds ✅
